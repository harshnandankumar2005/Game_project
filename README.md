## 🎮 How to Play

### Getting Started

1. **Clone or Download** the repository
2. **Open `index.html`** in your web browser
3. **Select Your Character** from the menu
4. **Battle Your Way** through enemy waves!

### Controls

```
ARROW KEYS  → Move around the arena
SPACEBAR    → Attack/Strike
Q KEY       → Activate Special Ability
ESC         → Pause Game
```

### Game Objective

- 🎯 **Survive** against waves of enemies
- 💪 **Defeat** all enemies in each wave
- 🏆 **Score Points** for kills and combos
- ⬆️ **Increase Difficulty** and challenge yourself
- 👑 **Reach the Top** of the leaderboard!

## ⚙️ Game Mechanics

### Combat System

| Aspect | Description |
|--------|-------------|
| **Health** | Each character has HP, reduced by enemy attacks |
| **Damage** | Attack damage varies by character & difficulty |
| **Cooldowns** | Special abilities have recharge times |
| **Knockback** | Successful hits create tactical spacing |
| **Combos** | Chain attacks for score multipliers |

### Difficulty Progression

```
Wave 1   → 🟢 Easy (5 enemies)
Wave 2   → 🟡 Medium (8 enemies)
Wave 3   → 🟠 Hard (12 enemies)
Wave 4+  → 🔴 Nightmare (∞ enemies)
```
